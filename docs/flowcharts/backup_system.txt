BigEd CC — Backup System Flow
================================

[Triggers]
        |
        +---> Auto-save timer (every 20min default, min 3min)
        +---> Fleet startup (supervisor.py boot)
        +---> Manual: Settings --> Operations --> "Backup Now"
        +---> CLI: lead_client.py backup
        +---> Skill deploy completion (planned)
        |
        v
[BackupManager.perform_backup()]
        |
        +---> Create timestamped dir: ~/BigEd-backups/YYYYMMDD_HHMMSS/
        |
        v
[Step 1: WAL Checkpoint]
        |
        +---> PRAGMA wal_checkpoint(TRUNCATE) on fleet.db
        +---> PRAGMA wal_checkpoint(TRUNCATE) on rag.db
        +---> Ensures all writes are flushed to main DB file
        +---> Prevents backing up incomplete transactions
        |
        v
[Step 2: Copy Targets]
        |
        +---> fleet.db -----> backup_dir/fleet.db (tasks, agents, usage)
        +---> rag.db -------> backup_dir/rag.db (knowledge embeddings)
        +---> fleet.toml ---> backup_dir/fleet.toml (config)
        +---> knowledge/ ---> backup_dir/knowledge/ (all artifacts)
        |
        +---> Each file: SHA-256 hash computed for manifest
        |
        v
[Step 3: Integrity Verification]
        |
        +---> PRAGMA integrity_check on backup/fleet.db
        +---> PRAGMA integrity_check on backup/rag.db
        +---> Results: "ok" or "FAILED" per database
        |
        v
[Step 4: Write Manifest]
        |
        +---> manifest.json:
        |       id, timestamp, trigger, location
        |       files: {name, size_bytes, sha256}
        |       integrity: {fleet.db: "ok", rag.db: "ok"}
        |       total_size_mb, duration_secs
        |
        v
[Step 5: Prune Old Backups]
        |
        +---> prune_enabled? --No--> Skip (infinite backup mode)
        |         |                     |
        |         |                     +--> Disk warning if > 80% used
        |         |
        |         Yes
        |         |
        +---> List backup dirs (sorted newest first)
        +---> Keep first N (depth, default 10, max 20)
        +---> Delete oldest beyond depth
        |
        v
[Complete]
        |
        +---> Log: "Backup complete: ID (X.X MB)"
        +---> Return manifest dict

[Settings UI]
        |
        +---> Operations tab --> "Auto-Save & Backup" section
        +---> Interval dropdown: 3/5/10/15/20/30/60 min
        +---> Depth dropdown: 1-20
        +---> "Keep all backups" checkbox (do not clean toggle)
        |       Warning: "backups will grow indefinitely"
        +---> Location: ~/BigEd-backups/ + Change button
        +---> "Backup Now" button (manual trigger)

[Graceful Shutdown]
        |
        +---> _graceful_save_tasks():
        |       RUNNING tasks --> PENDING (resume on next boot)
        |       All agents --> OFFLINE
        |       Shutdown marker note in DB
        +---> _unload_all_ollama_models():
        |       keep_alive=0 on all models (free VRAM)
        +---> _zombie_sweep():
                Kill any orphaned fleet processes
